# SpartsPro

Лендинг компании по поставкам лифтового оборудования, комплектующих и запчастей.

## Стек

- HTML5
- CSS3 (без фреймворков)
- Vanilla JS

## Структура проекта

```
van-gogh-link-test/
├── fonts/          # Шрифты Roboto и Open Sans
├── images/         # Фотографии продуктов и компании
├── index.html      # Основная страница
├── index.js        # Логика бургер-меню
└── styles.css      # Стили и адаптив
```

## Адаптивность

| Брейкпойнт | Устройство        |
|------------|-------------------|
| 1440px     | Десктоп           |
| 1160px     | Планшет широкий   |
| 768px      | Планшет           |
| 576px      | Мобильный         |
| 390px      | Мобильный малый   |

## Запуск

Открыть `index.html` в браузере или использовать Live Server в VS Code.
